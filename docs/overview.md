# Project Overview

This project is a simple console-based To-Do List application built using Python.  
It allows users to add, view, and remove tasks through a menu-driven interface.

The goal of the project is to demonstrate:

- Basic Python programming concepts
- Input validation and error handling
- Use of Visual Studio Code as a development environment
- Version control using Git
- Hosting a project on GitHub
- Proper project documentation following software development practices
